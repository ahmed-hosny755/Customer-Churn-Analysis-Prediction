# Customer Churn Analysis & Prediction

**Internship Project — Saiket Systems**

Brief: Analyze telecom customer churn and build predictive models to identify at-risk customers.

## Repository structure
See folders: `notebooks/`, `src/`, `images/`, `data/`.

## How to run
1. Create virtual env: `python -m venv venv`
2. Activate & install: `pip install -r requirements.txt`
3. Open notebook: `jupyter notebook notebooks/churn_analysis.ipynb`
